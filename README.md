# Anon-Ecommerce-Website-clone

A fully responsive **Ecommerce Website Clone** built using **HTML, CSS, and JavaScript**. This project replicates the design and functionality of an online shopping platform with features like product listing, cart management, and a user-friendly interface.

## 🚀 Features

- **Responsive Design** – Fully optimized for all devices (mobile, tablet, desktop).
- **Product Listing** – Displays items dynamically with images, descriptions, and prices.
- **Shopping Cart** – Add, remove, and update cart items with JavaScript.
- **Search & Filter** – Easily search and filter products.
- **User-friendly UI** – Smooth navigation and interactive elements.

## 🛠️ Technologies Used

- **HTML** – Structuring the website layout.
- **CSS** – Styling and responsive design.
- **JavaScript** – Implementing interactivity and dynamic features.

## 📂 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-username/anon-ecommerce-clone.git
   ```
2. **Navigate to the project folder**:
   ```bash
   cd anon-ecommerce-clone
   ```
3. **Open `index.html` in a browser** to view the project.
